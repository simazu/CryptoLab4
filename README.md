# Алгоритм хэширования MD4
### Реализованы следующие функции:
1. Вычисление хэша введенной строки
2. Проверка лавинного эффекта 
3. Поиск коллизии
4. Поиск прообраза
5. Вычисление некоторых зависимостей и построение графиков по ним
